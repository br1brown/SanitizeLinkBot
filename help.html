Sanitize Link - Guida rapida Chat privata Incolla uno o più link: ricevi la versione pulita, senza parametri di tracciamento e con l’URL finale. Nei gruppi Rispondi a un messaggio che contiene link e menziona il bot ({mention_bot}). Invio i link sanificati di quel messaggio. Ricerca inline Scrivi {mention_bot} nella barra di testo di Telegram. In modalità inline viene sanificato solo il primo link rilevato. Cosa faccio esattamente - Seguo i redirect fino alla destinazione finale (HTTP, meta refresh, redirect JavaScript comuni) - Rimuovo frammenti non necessari e parametri di tracciamento (esempio: utm, fbclid, gclid) - Controllo che il link pulito sia raggiungibile prima di inviarlo Dettagli utili - I link tipo mailto: e tel: non vengono modificati - Nei gruppi è necessario rispondere menzionando il bot al messaggio che contiene i link - Se ci sono molti link l’anteprima è disattivata per migliorare la leggibilità Codice sorgente: GitHub
